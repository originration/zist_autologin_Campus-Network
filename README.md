# zist_autologin_Campus-Network
本插件能够自动登录中原科技校园网

下载链接：https://wwop.lanzoul.com/ipREZ39wk1sj

开源链接：https://github.com/originration/zist_autologin_Campus-Network

图文教程：https://ycno9dy2hur8.feishu.cn/wiki/HRrQwMAqXidgE3kYTQecy2vknBU?from=from_copylink

理论上只要是drcom的认证计费系统都可以使用，但是每个学校的认证地址和认证界面不一样，可按需更改。

煮波是许昌校区的，没见过郑州校区的界面，不确定能不能用

原理：根据定位账号密码输入框、选择框和登录按钮的完整路径XPATH，使用js操作进行自动填充和点击，并带有信息保存和错误后自动重试功能。

理论上windows每次开机如果没有登录会自动弹出浏览器进入登录页面，借助此插件可以弹出后可直接登录

[这是python版本的][(GitHub - originration/zist_autologin_python)](https://github.com/originration/zist_autologin_python)，里面有登录原理解析，感兴趣的同学可以研究一下

使用教程：

1、打开浏览器（以Edge为例），点击右上角的选项➡️插件➡️管理扩展

2、打开开发者模式

3、下载压缩包并解压

4、点击“加载解压缩的扩展”。选中解压的文件夹

6、点击浏览器右上角扩展选项，置顶插件并点击

7、填写自己的信息点击保存

8、浏览器地址栏输入172.22.5.2测试是否正常登录。
